Web Dev Club Website ~ Spring 2022

We started this website to represent our club and have a bigger reach to the UVU community and the developer community.
This is our first design of the website. Our first step was brainstorming the site map, then we made low-fidelity wireframes for each page using Balsamiq. After agreeing upon the design, a Figma high-fidelity was created and that was used by the developers to create the website pages. Our website is going to be constantly evolving to try out new technologies and give members experience working on group projects with Github.

Want to get involved? Here are some future features that we want to implement!

  • Create responsive mobile, tablet and desktop webpages
  • Convert the website to a CMS
  • Create a carousel for the projects page
  • Create a contact form
  • Implement Sass
